第一模块

IP定位体系

核心定位 · 三圈模型

能力圈 · 你能给什么

你在美业/医美/抗衰领域的核心专业壁垒。不是"做了很多年"，而是有哪个具体问题是你能解决、别人解决不了的。定位从这里出发。

用户圈 · 她需要什么

35-50岁高净值女性用户的核心焦虑：变老、失控、被割韭菜、不被看见。你的IP要精准接住这种焦虑，并给出专业出口。

市场圈 · 空白在哪里

找到竞品没有占据的位置：科普类太学术、网红类太浮躁、机构类太销售。你的定位区间在"专业+真实+高价值感"的交叉地带。

人设标签系统

核心标签（只选一个）

  • 「抗衰顾问」— 专注30+女性逆龄管理，不贩卖焦虑，给真方案

  • 「美业创始人说」— 一个开了XX年机构的老板，讲行业真话

人设辅助标签（3-5个）

  • 身份背书标签：执业资质 / 机构年限 / 专业学历
  • 性格标签：真实 / 不打马虎眼 / 敢说同行不敢说的话
  • 生活标签：精致妈妈 / 独立女性 / 自我管理践行者
  • 视觉标签：固定穿搭色系 + 固定拍摄场景（诊所/书房/化妆台）

信任感设计路径

专业信任

  • 定期露出资质/证书/机构环境
  • 使用专业词汇但立即翻译成人话
  • 引用有来源的数据/研究
  • 勇于说"这个我不确定"

真实信任

  • 展示工作状态的"不完美切面"
  • 分享自己踩过的坑和后悔的决定
  • 不回避争议话题，有立场有态度
  • 评论区真实互动，不删差评

价值信任

  • 每条视频给出一个"带走即可用"的干货
  • 主动反推销：帮用户省钱而不是让她消费
  • 长期一致的价值观输出，不随热点漂移
  • IP形象全平台统一，认知不分裂

内容世界观 · 长期方向规划

内容世界观宣言（示例框架）

  • 核心信念：「真正的抗衰是让自己越活越值钱，不是对抗岁月，是驾驭岁月」

长期内容方向 · 主线

长期内容方向 · 辅线

5-15秒

核心价值

情绪收尾
50-60秒
价值感句子 + 轻引导：「聪明的女人应该把钱花在刀刃上——关注我，我帮你做那把刀」
评论钩子
可选

美业口播五大内容类型

女性情绪价值型

  • 核心逻辑：先共情情绪，再输出认知，最后给力量感
  • 例：「35岁之后，很多女人开始觉得自己变丑了。但我见过太多客户，40岁比25岁更美——因为她学会了管理自己。」
  • 配套情绪：被看见 / 被认可 / 有力量 / 不孤单

专业科普拆解型

  • 核心逻辑：把专业知识降维到"小学生听得懂"的程度
  • 结构：现象（你看到的）→ 真相（实际是什么）→ 行动建议
  • 禁忌：不要一条视频讲超过3个知识点

行业真相揭秘型

  • 核心逻辑：利用信息差，创造"只有内行才知道"的优越感
  • 例：「你花1000块做的XX项目，成本只有80」/ 「医美机构最怕你知道的事」
  • 注意：揭秘行业问题 ≠ 攻击同行，要上升到"帮用户做正确选择"

成交型内容逻辑

  • 核心逻辑：在内容中埋植"决策锚点"，让用户自发产生咨询意愿
  • 结构：问题放大 → 方案对比 → 正确路径 → 行动指引（轻）
  • 关键：不在视频里说"加我""联系我"，要让用户自己找过来

系列栏目设计（建立账号记忆点）

固定栏目 · 主力

  • 「真相局」— 每周一条，颠覆一个美业认知
  • 「创始人日记」— 每周一条，展示真实工作切面
  • 「30秒必知」— 超短型，一个知识点，高完播率

周期性栏目 · 辅助

  • 「本周我见过的案例」— 匿名故事化处理，真实感强
  • 「问答实录」— 用评论区提问直接做内容，互动数据极好
  • 「选什么别选什么」— 对比型，极易上热门

选题系统 · 来源

  • 来源①：客户咨询时高频问题（最接地气）
  • 来源②：平台热门评论区的焦虑词
  • 来源③：竞品爆款评论区"没被回答的问题"
  • 来源④：行业内最近出现的争议/新项目/新说法
第三模块

成片与账号体系

Post-Production · Visual Identity · Account Consistency

剪辑策略 · 节奏设计

剪辑节奏规则

  • 口播片段：每2-3秒一个切点（无实质内容的停顿全部剪掉）
  • 第一个字要"跳入画面"，不能有任何前摇
  • 关键信息处：切为大字幕全屏特写，强化记忆点
  • 结尾前0.5秒：加一个视觉或声音的「收束动作」
  • 全片不超过70秒（首月建议40-60秒）

字幕与文字策略

  • 字幕字号：正文22-24px，强调词放大至32px以上
  • 关键词加色标注（品牌色，全账号统一）
  • 每5-8秒出现一个「视觉锚点」（小标题/表情包/数据卡片）
  • 禁止密集字幕覆盖全屏——留呼吸空间
  • AI字幕全部校对，错别字是专业度杀手

高级感视觉方向

色彩基调

确定2-3色品牌色。医美偏冷白/莫兰迪/深灰金；抗衰偏米杏/玫瑰金/奶油白；皮肤管理偏奶茶色/裸感。全账号不换色系。

字体品牌感

全片使用1-2款字体，不超过3种。主标题：有设计感的中文黑体/衬线。正文：清晰可读的细黑体。禁止系统默认黄色字幕。

视觉锁定点

找到一个"只要看到这个元素，就知道是你"的视觉钩子：固定开场动态字卡、固定片尾Logo动效、固定角标品牌标识。

B-roll 规划思路

B-roll类型优先级

  • P1 — 环境B-roll：诊所/操作间/工作台，建立专业感（每月固定素材库）
  • P2 — 产品/设备B-roll：仪器特写、产品细节，配合讲解使用
  • P3 — 人文B-roll：工作中的手部/局部动作，无需跟拍即可制作
  • P4 — 数据可视化B-roll：用动态图表替代纯讲数据

B-roll素材自建指南

  • 每月集中拍摄1次，积累30-50条素材（手机即可）
  • 固定场景：操作台 / 诊室走廊 / 产品陈列 / 书桌工作
  • 固定道具：品牌马克杯 / 专业手册 / 白大褂 / 设备特写
  • 不需要跟拍，所有B-roll自主可控，降低制作成本

封面标题系统 · 账号统一性

封面设计规则

  • 封面字数控制在8字以内（最多12字）
  • 固定版式模板（1-2套），换内容不换版式
  • 主色+白字 或 白底+主色字，两套方案切换
  • 主图：真人出镜 > 产品特写 > 设计封面

标题写作公式

  • 数字型：「3个信号说明你的胶原蛋白已经流失了」
  • 反常识型：「越贵的护肤品越可能毁掉你的皮肤屏障」
  • 身份型：「医美机构不告诉你的7件事」
  • 对话型：「你做光子嫩肤之前应该先问这3个问题」

账号统一性检查清单

  • 头像：真人形象，光线稳定，表情有力量感
  • 昵称：含核心标签词，简洁可记忆（最多8字）
  • 简介：一句话定位 + 核心价值 + 轻引导动作
  • 内容：翻开前6条，视觉色调/字体/框架是否统一

信任钩子

简介第一行是专业背书；主页置顶1条"品牌故事/自我介绍"视频

小红书

定位

搜索流量核心阵地，种草逻辑，图文+短视频双轨并行

起号逻辑

前30篇全部布局SEO关键词；发布时间固定，账号权重更稳定

流量打法

标题含精准搜索词（"医美避坑"/"水光针选择"）；首图决定点击率

评论区

每条笔记回复前10条评论；用"你可以私信我"代替外部跳转

信任钩子

高光笔记集：「看完这一篇，足够做决定」型内容做合辑

视频号

定位

微信生态内的信任放大器，偏中老年/本地/熟人传播逻辑

起号逻辑

同步发布抖音内容；利用朋友圈分享放大初始流量；标签精准

流量打法

视频号重「转发」逻辑——内容要能让粉丝有"我要分享给朋友"的冲动

评论区

评论区积极置顶重要内容；适当引导微信好友添加

信任钩子

与公众号联动：长内容公众号+短视频视频号，形成生态闭环

用户转化路径设计（内容前端层）

前端内容的转化漏斗：看见 → 认知 → 信任 → 主动触达

  • 第一层「看见」：靠HOOK抓住注意力，前3秒决定完播率
  • 第二层「认知」：靠内容质量建立"这个人说的比别人更准"的心智认知
  • 第三层「信任」：靠系列内容和评论区互动，建立"她是真的懂我"的情感连接
  • 第四层「主动触达」：不是你去找用户，是用户主动来找你——评论留言/点击主页/主动私信
  • 内容前端的终点：让用户产生「我想找这个人聊聊」的冲动

核心数据指标体系（内容前端）

流量质量指标

  • 完播率（>35% = 内容及格，>50% = 优质）
  • 粉丝增长率（周同比/月同比）
  • 主页访问量（反映封面+标题吸引力）
  • 自然流量比（判断账号标签是否精准）

内容质量指标

  • 互动率（点赞+评论+收藏/播放量）
  • 收藏率（>3% = 内容有实用价值）
  • 评论质量（是否出现"想了解更多"）
  • 转发分享率（视频号/小红书重点关注）

人设健康指标

  • 粉丝画像是否与目标客群吻合
  • 评论区"内容调性"词云是否正向
  • 是否出现"专业、真实、有用"类评价
  • 高客单用户主动私信/问询数量

阶段升级路径（6个月规划）

第1-2月：账号定型期

  • 核心任务：跑通内容节奏，稳定发布频次（4-5条/周）
  • 内容策略：纯垂直内容，不追热点，打算法标签
  • 成功标志：账号标签准确，有条视频完播率超过40%
  • 人设任务：建立"专业且真实"的基础认知

第3-4月：流量放大期

  • 核心任务：复制爆款内容模型，提升账号权重
  • 内容策略：80%稳定内容 + 20%热点借势，有策略地追流量
  • 成功标志：月均播放量稳步增长，主动咨询用户出现
  • 人设任务：推出系列栏目，建立账号记忆符号

第5-6月：品牌强化期

  • 核心任务：IP形象全面成型，开始跨平台联动
  • 内容策略：开始布局较长内容 / 专业合集 / 深度系列
  • 成功标志：被用户主动推荐 / 出现"找这个人看看"的自然传播
  • 人设任务：确立行业权威IP地位，高客单用户信任闭环建立

持续迭代原则

  • 每月至少迭代1个内容结构或选题方向
  • 每季度重新审视IP定位是否需要微调
  • 爆款内容：3个月内二次深挖，围绕同话题拓展系列
  • 账号数据下滑时：先回归基本面，不要盲目转型
